# Bike Rental Demand — Streamlit Demo

This repository contains a compact Streamlit demo that predicts hourly bike rental demand using a trained model pipeline.

Files
- `app.py` — Streamlit single-file demo (UI, inputs, model loading, visualization).
- `models/bike_model.pkl` — Trained model pipeline (required to run predictions).
- `requirements.txt` — Python dependencies (install before running).
- `images/screenshot.svg` — Placeholder screenshot of the running app (replace with a real PNG/JPEG for the final README).
